# BarcodeAutoReader
 - 단말기에 설치되어 기계장치와 바코드 리더기계와 통신하며 제어하는 프로그램입니다.
 
## 하드웨어 구성

## 소프트웨어 요구사항
 - [ ] 바코드 리더기로부터 바코드정보를 받음 
 - [ ] 받은 바코드정보를 저장함 (파일 또는 Excel, 각각 파일은 날짜 단위로 나눔)
 - [ ] 기계장치와의 통신 구현: 기계장치에게 전송(시작해라, 바코드 읽음)
 - [ ] 기계장치와의 통신 구현: 기계장치로부터 수신(바코드 읽어라)
